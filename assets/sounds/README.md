# Animal Sound Effects

This folder should contain audio files for animal persona sound effects.

## Required Files:

- `bark.mp3` - Dog bark sound
- `pant.mp3` - Dog panting sound
- `meow.mp3` - Cat meow sound
- `purr.mp3` - Cat purring sound
- `roar.mp3` - Lion roar sound
- `growl.mp3` - Lion/animal growl
- `monkey.mp3` - Monkey chatter
- `oink.mp3` - Pig oink sound
- `heehaw.mp3` - Donkey sound

## Where to get free sound effects:

1. https://freesound.org (Free, requires attribution)
2. https://pixabay.com/sound-effects (Free, no attribution)
3. https://mixkit.co/free-sound-effects (Free)

## Format Requirements:

- MP3 or WAV format
- Short clips (1-3 seconds)
- Mono or stereo, any sample rate
